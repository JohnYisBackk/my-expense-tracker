# Expense Tracker

A modern, responsive expense tracker built with JavaScript. Includes animated totals, undo feature, budget limit bar, dark mode, and category stats with charts.  

## Features
- **Add / Edit Transactions** – Income and expense entries with name, category, amount, and date.
- **Animated Totals** – Balance, income, and expenses animate dynamically.
- **Undo Last Transaction** – 5-second timeout to revert accidental deletions.
- **Budget Limit Bar** – Visual percentage indicator for expenses vs budget.
- **Filter Transactions** – Filter by category or type.
- **Dark Mode** – Toggle between light and dark themes.
- **Category Stats** – Pie chart for expenses, line chart for balance over time.

## Built With
- HTML5 & CSS3
- JavaScript (ES6)
- Chart.js for charts
