# Expense Tracker
A modern, responsive expense tracker with animated totals, undo feature, budget limit bar, dark mode, and category stats with charts.

## Features
- Add income and expense transactions
- Animated total balance, income, and expenses
- Undo last transaction (5-second timeout)
- Budget limit bar with percentage indicator
- Filter transactions by category or type
- Dark mode toggle
- Category stats with pie and line charts

## Built With
- HTML5 & CSS3
- JavaScript (ES6)
- Chart.js for charts
